# 🏥 Hospital Management System (Python GUI)

## 📌 Description
This is a **Python-based Hospital Management System** with a **Graphical User Interface (GUI)**.
The project allows users to **add patients** and **remove patients** easily using a simple GUI.
It is specially made for **students and beginners**.

---

## ✨ Features
- Add patient records  
- Remove patient records  
- Simple and user-friendly GUI  
- Beginner-friendly Python code  

---

## 🛠 Requirements
- Python 3.x  
- No advanced libraries required  
- Works on Windows, macOS, and Linux  

---

## ▶ How to Run
1. Install Python 3.x  
2. Download and extract the project ZIP file  
3. Open the project folder  
4. Run the main file:
   ```bash
   python main.py
   ```
5. The GUI window will open  

---

## 🎯 Use Case
- School / college projects  
- Python GUI practice  
- Learning basic hospital management logic
